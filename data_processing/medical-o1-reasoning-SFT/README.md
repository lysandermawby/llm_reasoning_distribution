# medical-o1-reasoning-SFT

[medical-o1-reasoning-SFT](https://huggingface.co/datasets/FreedomIntelligence/medical-o1-reasoning-SFT) is a set of GPT-4o-generated responses to medical questions. These are mostly phrased as situation and symptom descriptions.

Note that this dataset does not show CoT as a separate output.
