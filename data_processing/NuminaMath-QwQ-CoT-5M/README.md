# NuminaMath-QwQ-CoT-5M

[NuminaMath-QwQ-CoT-5M](https://huggingface.co/datasets/PrimeIntellect/NuminaMath-QwQ-CoT-5M) is a dataset of QwQ responses to mathematical problems.

Note that this is not structured to contain CoT data in clear thinking tags. All text is treated as non-thinking in this analysis.
